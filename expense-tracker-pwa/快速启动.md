# 🚀 快速启动指南

## 第一步：安装依赖

```bash
cd expense-tracker-pwa
npm install
```

这会安装所有必要的依赖包，大约需要 1-2 分钟。

## 第二步：启动开发服务器

```bash
npm run dev
```

看到以下输出表示启动成功：

```
  VITE v5.0.8  ready in 500 ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
  ➜  press h to show help
```

## 第三步：打开浏览器

在浏览器中访问：`http://localhost:5173`

## 第四步：初始化数据

1. 点击底部导航栏的"设置"
2. 点击"初始化"按钮
3. 系统会自动创建：
   - 2个归属人（男主、女主）
   - 31个支出类型
   - 3个收入类型  
   - 2个不计入类型
   - 10个支付方式（每人5个）

## 第五步：开始使用

1. 点击"账单"标签
2. 点击"添加账单"按钮
3. 填写账单信息并保存
4. 查看统计数据

## 🎉 完成！

现在你可以：
- ✅ 添加、编辑、删除账单
- ✅ 使用筛选功能
- ✅ 查看统计数据
- ✅ 导出数据为 CSV
- ✅ 导入 CSV 数据

## 📱 安装到手机

### iOS
1. 用 Safari 打开应用
2. 点击分享按钮
3. 选择"添加到主屏幕"

### Android
1. 用 Chrome 打开应用
2. 点击菜单（三个点）
3. 选择"安装应用"

## 🏗️ 构建生产版本

```bash
npm run build
```

构建产物在 `dist` 目录，可以部署到任何静态服务器。

## 🐛 常见问题

### 端口被占用
如果 5173 端口被占用，Vite 会自动使用下一个可用端口。

### 依赖安装失败
尝试清除缓存：
```bash
rm -rf node_modules package-lock.json
npm install
```

### 页面空白
1. 检查浏览器控制台是否有错误
2. 确保已初始化数据
3. 尝试清除浏览器缓存

## 💡 开发提示

- 修改代码后会自动热更新
- 按 `Ctrl+C` 停止开发服务器
- 数据保存在浏览器 IndexedDB 中
- 清除浏览器数据会删除所有账单
