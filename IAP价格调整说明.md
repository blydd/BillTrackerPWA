# IAP 价格调整说明

## 价格修改

### 调整内容
- **年订阅**: ¥18/年 → ¥12/年 (降价 ¥6)
- **终身买断**: ¥43 → ¥40 (降价 ¥3)

### 修改原因
- 优化定价策略，提高用户接受度
- 降低付费门槛，增加转化率
- 保持合理的价格差异，鼓励长期使用

## 修改的文件

### 1. Products.storekit
```json
// 年订阅价格
"displayPrice" : "12"  // 原来是 "18"

// 终身买断价格  
"displayPrice" : "40"  // 原来是 "43"
```

### 2. Services/IAPManager.swift
```swift
var displayPrice: String {
    switch self {
    case .annualSubscription: return "¥12/年"  // 原来是 "¥18/年"
    case .lifetimePurchase: return "¥40"       // 原来是 "¥43"
    }
}
```

## 价格策略分析

### 新定价优势
1. **年订阅 ¥12/年**:
   - 月均 ¥1，心理价位更容易接受
   - 与市面上同类应用价格更有竞争力
   - 降低用户首次付费门槛

2. **终身买断 ¥40**:
   - 相当于 3.33 年的年订阅费用
   - 对长期用户更有吸引力
   - 整数定价更简洁

### 价格对比
| 购买方式 | 原价格 | 新价格 | 降幅 | 备注 |
|---------|-------|-------|------|------|
| 年订阅 | ¥18/年 | ¥12/年 | -33% | 月均 ¥1 |
| 终身买断 | ¥43 | ¥40 | -7% | 约 3.33 年回本 |

### 用户选择建议
- **短期用户**: 选择年订阅，成本更低
- **长期用户**: 选择终身买断，一次付费永久使用
- **试用用户**: 可先选择年订阅体验，后续升级到终身版
