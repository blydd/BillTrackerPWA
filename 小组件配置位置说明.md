# 小组件配置位置说明

## 📱 在应用中的位置

### 导航路径：
```
1. 打开 ExpenseTracker 应用
2. 点击底部的 "设置" 标签页 (⚙️ 图标)
3. 滚动到 "快速功能" 部分
4. 点击 "小组件配置"
```

## 🎯 设置页面结构

在设置页面中，您应该看到以下部分：

### 1. 订阅状态
- 显示当前的 Pro 订阅状态
- 升级按钮（如果是免费用户）

### 2. 快速功能 ⭐
- **小组件配置** ← 这里就是您要找的！

### 3. 数据管理
- 账单类型管理
- 归属人管理
- 支付方式管理
- 数据库导出

### 4. 系统
- 初始化
- 数据库信息

## 🔍 如果找不到"小组件配置"

### 可能的原因：
1. **应用没有正确编译**
   - 重新编译项目 (Command + B)
   - 确保没有编译错误

2. **代码没有正确更新**
   - 检查 ExpenseTrackerApp.swift 文件
   - 确保包含 SimpleWidgetConfigView

3. **模拟器缓存问题**
   - 重新运行应用
   - 或者重置模拟器
